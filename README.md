# Automatic tool to install Arch Linux and a custome awesome theme

## First step install git
```
pacman -Sy git
git clone https://github.com/LordGuccif3r/AwesomeArch.git
cd AwesomeArch
chmod +x Auto_Arch.sh
./Auto_Arch.sh
```
